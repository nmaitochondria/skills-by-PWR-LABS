# skills-by-PWR-LABS

Public home for **[PWR] LABS** [Cursor Agent Skills](https://cursor.com/docs/context/skills): portable **`SKILL.md`** instructions used across the portfolio.

## Notion (portfolio hub)

**[skills by PWR LABS — Notion](https://www.notion.so/skills-by-PWR-LABS-35d50bc778328020a178d10e7a091bee)** — linked database, one row per catalog skill; seeded from git via [PWR-LABS/ribosome](https://github.com/PWR-LABS/ribosome) `scripts/seed_pwr_skills_notion_pages.py`.

## Git vs this repo

Canonical **`SKILL.md`** files still live under each product checkout (for example `ribosome/.cursor/skills/…`, `chaperone/.cursor/skills/…`). This repository is the **skills-only mirror** for shareable paths and raw links. Keep folder names aligned with YAML **`name:`** and **`docs/SKILLS_PUBLICATION_CATALOG.md`** in **ribosome**.
