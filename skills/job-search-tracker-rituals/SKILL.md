# Run job-search tracker rituals

## What this unlocks

Turns a long job search into **repeatable operator rituals**: weekly review, tracker hygiene, tailoring prep, and interview prep—without pretending the model is an oracle.

## Origin in the [PWR] ecosystem

Anchors **[ribosome.]** career-transition posture: Notion as system of record, bounded LLM assistance, and explicit next actions for clinician and health-professional pivots (and industry-agnostic reuse).

## Broader applications

- Any professional running applications across many employers with one canonical tracker.
- Coaches who want a shared checklist vocabulary with clients.
- Internal mobility programs that need discipline more than more job alerts.

## When to use

User is managing applications in Notion (or similar), wants ritual discipline: weekly review, draft tailoring prep, follow-ups — **without** clinician-specific narrative yet.

## Practices

1. **Single system of record** — one tracker DB; avoid duplicate spreadsheets.
2. **Weekly review** — filter rows touched this week; count applies, networking touches, interviews, outcomes.
3. **Draft hygiene** — job URL + resume variant noted on row before generating outreach.

## Outputs

- Short weekly summary bullets (counts + blockers).
- Prompt stubs for tailoring a paragraph to a posting (no MD-specific vocabulary unless user asks).
