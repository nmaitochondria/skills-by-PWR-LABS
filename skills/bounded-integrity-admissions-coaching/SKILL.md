# Coach admissions (bounded integrity)

## What this unlocks

Defines a **non-negotiable integrity contract** for coaching: human owns facts, tooling owns structure, and the model never becomes a dishonest ghostwriter or admissions oracle.

## Lane context

**[chaperone.]** core lane posture—grad, med, and other professional lanes with explicit vocabulary for boundaries and revisable drafts.

## Broader applications

- Mentorship programs in law, business, or clinical training with similar integrity constraints.
- Any AI-assisted writing where “confidence” must not substitute for verified claims.
- Policy templates for edtech vendors serving regulated student populations.

## When to use

- Any **[chaperone.]** workflow where the model could **smooth uncertainty** into overconfident prose or **invent** facts.
- Choosing which **lane** vocabulary to use (grad vs med vs other professional).

## Operator contract (non-negotiable)

- **Human owns facts:** grades, activities, timelines, program-specific requirements — verified by applicant + coach.
- **Tooling owns structure:** checklists, versioning, Notion state, automation that **surfaces gaps** without inventing content.
- **What we are not:** an admissions oracle, a ghostwriter for dishonest claims, or a generic chatbot replacing school-specific research.

## Lanes (intent vocabulary)

| Lane | Notes |
| --- | --- |
| **Grad (non-med)** | Research fit, POI outreach, writing-heavy programs. |
| **Med / MD / DO** | AMCAS/TMDSAS constraints, secondaries volume, ethical boundaries on narrative. |
| **Other professional** | Law, dental, etc. — add **separate playbooks** when needed; do not bloat core doctrine. |

## Outputs the agent should prefer

- **Explicit constraints** (“answer this prompt only,” “cite what you verified”).
- **Revisable drafts** with visible assumptions and **integrity** callouts where claims need human confirmation.
